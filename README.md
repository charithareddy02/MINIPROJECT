# Vulnerability Analysis of Urban Slums for Disaster Preparedness
## Abstract

The Vulnerability Analysis of Urban Slums for Disaster Preparedness is a web-based platform designed to identify and monitor disaster-prone urban slum areas. The system integrates real-time weather monitoring, vulnerability mapping, risk analytics, emergency response services, and AI-powered assistance to improve disaster preparedness and public safety.
Developed using HTML, CSS, JavaScript, FastAPI, Python, and SQLite, the platform visualizes high, medium, and low-risk zones on interactive maps. It provides weather-based alerts, emergency contact information, disaster history records, and risk assessment insights to help users make informed decisions during emergencies.

## Technologies Used

- HTML
- CSS
- JavaScript
- Python
- FastAPI
- SQLite

## Features

- Vulnerability Mapping
- Disaster Risk Analysis
- Emergency Response Support
- Real-Time Weather Monitoring
- Disaster History Tracking
- AI Assistant
